My Artwork

Digital Composition

August 2020

This was the very first assignment in our Art 74 class. Our goal was to take 2 different photos/things and combine them using either Adobe Photoshop, Illustrator or both. My Digital Composition was done in Adobe Photoshop and my goal was to stick with the "vaporwave" vibe. I added the Initial D character and changed the colors and added a shadow so that he could blend in with the background more. As well as adding the text on the top.

3D MeshMixer Dragon x Angel

September 2020

This was a later assignment done in 3D Meshmixer. Our goal was to combine two different things together and make a new object then render it in Adobe 3D Dimensions. In my piece I decided to make it seem like those cool outdoor public sculptures you see in the downtown if a city. I took a pair of angel wings and attached it to an asian dragon using a "burnt metal" material.

Glitch Art

September 2020

This assignment was probably my favorite. That's because knowing that I can manupulate a photo with it's audio is so crazy. Each photo has it's own unique audio and adding effects made the whole process so fun. With my piece I decided to use Junji's manga art work and add on to it's scary feeling. In the future I would like to do more personal glitch art projects combined with photography.

Java Script Digital Illustration

October 2020

Our Java Script project was probably the most difficult one to do, my reasoning is because of the coordinate system. It was trial and error when it came to trying to find the right coordinate when trying to make my illustration. This one specifically wasn't so bad because the code was very repetitive. This illustration was meant to be a chill summer in Japan type of feel.

Java Script Self Portrait

October 2020

This is my interenet persona which is me stuffed in my green hoodie. As previously mentioned I had a fairly difficult time with Java Script, specifically the coordination system. I had a difficult time when it came to making a self portrait because it was endless trial and error trying to find the right point for the 2D primitives.
